只需三步，连接你的终端

1

安装 CLI

一条命令全局安装。内置网关开箱即用，无需额外部署服务器。

# 全局安装 CLI
$ npm i -g linkshell-cli
# 初始化配置
$ linkshell setup
# 检查环境
$ linkshell doctor
✓ All checks passed
2

启动桥接
